Quick Start

# Install
npm install create-ai-e2e --save-dev

# Initialize
npx create-ai-e2e init

# Scan your routes
npx create-ai-e2e scan src/

# Generate tests
npx create-ai-e2e gen

# Run tests
npx playwright test
